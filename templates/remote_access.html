how your data is stored and how the network works

how your data gets to you

The sardinetracker app you're using right now runs on a Raspberry Pi 4 sitting in Alaric's house. Your health data — every daily entry, lab result, medication, and note — lives in a SQLite database on that Pi and never leaves it.

When you open the app in your browser, your request travels through an encrypted Tailscale tunnel to an Oracle Cloud VM, which forwards it to the Pi. The Oracle VM can't read your data — it just passes encrypted traffic. The only external connection the app makes is to a public weather API to fetch UV index data, using just your city-level coordinates.

The document below is the full technical guide for how this infrastructure is set up. You don't need to understand any of it to use the app — it's here for transparency.
